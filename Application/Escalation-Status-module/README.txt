Escalation Module:
About:
The Escalation module is a java file which checks the time of all the alotted ticket, 
If the ticket time exceeds the allotted time, the status will then be change to escalated remarks. 

To run the escalation Module:
-Simply copy the folder in netbeans Project.

Prerequisites:
The Escalation requires you a java Plugin
a. mysql connectorj 
b. jdbc
c. mysql drivermanager
